#  Team Task Manager

A simple **Task Management System** built using **Node.js, Express, and MongoDB**.  
This project allows users to create, manage, and track tasks efficiently.


##  Features

- User authentication (JWT)
- Create, update, delete tasks
- Assign tasks to team members
- Track task status (Pending / In Progress / Completed)
- REST API backend
- MongoDB database integration


##  Tech Stack

- Node.js
- Express.js
- MongoDB Atlas
- Mongoose
- JWT (Authentication)
- dotenv
